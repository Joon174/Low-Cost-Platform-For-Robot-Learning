\doxysubsection{Detailed Description}
\mbox{\hyperlink{classenv_register_1_1_proof_of_concept_model}{Proof\+Of\+Concept\+Model}}. 

Creates an environment for one leg of the Robotic Platform The following class creates an instance of one robotic leg of the platform. The environment is mainly used for unit testing for validation. The rewards of this environment are designed for the accuracy following a specified trajectory input.
