\begin{DoxyNote}{Note}
The following is the default architecture for this agent\+: 

Layer 1\+: Linear, (in\+\_\+channels = environment = 128) -\/$>$ Re\+LU 

Layer 2\+: Linear, (in\+\_\+channels = 128, out\+\_\+channels = 128) 

Layer 3\+: Linear, (in\+\_\+channels = 128, out\+\_\+channels = number of actions available for agent)
