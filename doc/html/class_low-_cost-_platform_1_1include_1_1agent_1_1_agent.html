Detailed Description

Agent The Parent class which can be used to create instances of Agents using inheritance.

The class checks for CUDA cores to use for GPU computation and read JSON files for the hyperparameters of the model. The class contains frequently used parameters in Reinforcecment Learning such as: rewards, loss, epoch number, time step and frame index. A Debugger is instantiated in the class for debugging should the user feel the need to.
