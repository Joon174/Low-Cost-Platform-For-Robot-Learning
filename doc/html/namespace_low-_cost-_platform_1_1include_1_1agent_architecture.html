Note
The following is the default architecture for this agent:
Layer 1: Linear, (in_channels = environment = 128) -> ReLU
Layer 2: Linear, (in_channels = 128, out_channels = 128)
Layer 3: Linear, (in_channels = 128, out_channels = number of actions available for agent)
The actions for this model are evaluated using the Epsilon Greedy Strategy
